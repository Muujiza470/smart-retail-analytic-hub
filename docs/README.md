# Documentation

This folder contains the full written report and supporting documentation for the Smart Retail Analytics Hub project.

## Files

- `smart-retail-final-report.pdf`

The report includes:

- OLTP system design (3NF normalization)
- OLAP dimensional modeling (Star Schema)
- JSON Relational Duality View implementation
- Business use cases and architectural decisions
- Technical explanation of Oracle 23ai features used

All key diagrams are displayed in the main project README.
