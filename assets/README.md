# Assets

This folder contains visual diagrams and architecture screenshots for the Smart Retail Analytics Hub project.

## Included Visuals

- `oltp-erd.png`  
  Entity Relationship Diagram (3NF) for the transactional OLTP system including Customers, Orders, Products, Stores, and Inventory.

- `olap-star-schema.png`  
  Dimensional Star Schema design showing FACT_SALES connected to DIM_CUSTOMER, DIM_PRODUCT, DIM_STORE, and DIM_TIME.

These visuals are embedded in the main project README for presentation and architectural reference.
